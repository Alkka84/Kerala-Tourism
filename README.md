#  Kerala Tourism Website

A static tourism website showcasing the beauty, culture, and destinations of Kerala – *God’s Own Country*.  
This project is built using **HTML and CSS** and highlights different districts, tourist spots, and travel experiences in Kerala.

---

##  Features

-  Home page with Kerala overview
-  District-wise destination pages
-  Separate pages for tourist destinations
-  Image gallery showcasing Kerala’s beauty
-  Login and Signup pages
-  Responsive layout using CSS
-  Organized folder structure for easy maintenance

---

##  Technologies Used

- **HTML5**
- **CSS3**
- **Git & GitHub**
- **GitHub Pages** (for deployment)
